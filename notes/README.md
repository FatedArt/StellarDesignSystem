# Notes — Ringkasan Sesi & Keputusan

Folder ini berisi catatan percakapan dan keputusan teknis yang bisa dibaca di device mana pun lewat Git.

## Cara pakai di device lain

1. Clone atau pull repo: `git pull origin development`
2. Buka file di `notes/sessions/`
3. Untuk konteks proyek lengkap, lihat juga [`CONTEXT.md`](../CONTEXT.md) di root repo

## Aturan update

- Tambah file baru di `sessions/` per sesi penting (jangan timpa file lama).
- Nama file: `YYYY-MM-DD-judul-singkat.md`
- Setelah update besar, sinkronkan juga `CONTEXT.md` bila perlu.
